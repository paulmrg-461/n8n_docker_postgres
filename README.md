## Getting Started

1. **Clone the repository**  
   ```bash
   git clone https://github.com/paulmrg-461/n8n_docker_postgres.git
   cd n8n_docker_postgres
   ```
2. **Run on docker http://127.0.0.1:5678/**
```
sudo docker compose up --build
```
